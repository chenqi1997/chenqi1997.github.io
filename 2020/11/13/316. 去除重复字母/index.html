316. 去除重复字母

题目：

给你一个字符串 s ，请你去除字符串中重复的字母，使得每个字母只出现一次。需保证 返回结果的字典序最小（要求不能打乱其他字符的相对位置）。

注意：该题与 1081 https://leetcode-cn.com/problems/smallest-subsequence-of-distinct-characters 相同

示例 1：

示例 2：

提示：

  • 1 <= s.length <= 104
  • s 由小写英文字母组成

解法：单调栈⭐⭐⭐⭐⭐

这道题是从 Leetcode 402 移掉K位数字 那里过来的，重点想实践一下单调栈的思想。

题目的要求总结出来有三点：

  1. 要去重。
  2. 去重字符串中的字符顺序不能打乱 s 中字符出现的相对顺序。
  3. 在所有符合上一条要求的去重字符串中，字典序最小的作为最终结果。

在每次遍历的时候，用布尔数组 isInStack 记录栈中元素，达到去重的目的，保证栈中的元素都是没有重复的。

同时还要维护一个 count 数组，用来记录每个字符还剩下多少个，然后在每次遍历的时候都要把这个字符对应的count--。

在单调栈的思想中，我们应该维护一个单调递增的栈，从而保证最终栈内保存的字符串是字典序最小的。

  1. 每次遍历到一个元素首先要判断是否它已经存在在栈中了，如果是的话直接 --count 然后 continue ;
  2. 正常的单调栈的判断，不过要在 while 循环内注意一下 acdc 这种情况，如果要 pop 的元素的 count 为 0 说明后面就再也没有该字符，所以应该 break 跳出 while 循环。
  3. 要记住入栈的元素要讲 isInStack 置为 true 。
