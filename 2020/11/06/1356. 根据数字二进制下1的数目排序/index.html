1356. 根据数字二进制下1的数目排序

题目：

给你一个整数数组 arr 。请你将数组中的元素按照其二进制表示中数字 1 的数目升序排序。

如果存在多个数字二进制中 1 的数目相同，则必须将它们按照数值大小升序排列。

请你返回排序后的数组。

解法：位运算+自定义sort的cmp函数⭐⭐⭐

这道题其实是考察如何计算一个数的二进制中1的数量，最开始的想法是用一个 mask 不断 <<1 来计算：

不过想了想就很麻烦，不像是位运算的风格。后来参考了官方答案的想法，利用 n & (n - 1) 来不断消除最低位的 1， 这样只需要有多少个 1 循环多少次了。

最后需要根据每个数的 bitCount 当作 sort 函数的 cmp 函数。这里面要注意的是：sort 函数第三个参数 cmp 的调用要求不带 this 指针，对于类内函数，他们都有一个隐藏参数 this 指针，因此要么把 cmp 写在类外，要么加上 static 声明，static 声明后的成员函数不带有 this 指针。所以这里面的两个函数 bitCount 和 cmp 都要是static 的。
