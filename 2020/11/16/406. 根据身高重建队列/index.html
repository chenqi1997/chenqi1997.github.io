Leetcode_406 根据身高重建队列

发表于 2020-11-16 | 分类于 LeetCode ， Medium

题目：

假设有打乱顺序的一群人站成一个队列。 每个人由一个整数对(h, k)表示，其中h是这个人的身高，k是排在这个人前面且身高大于或等于h的人数。 编写一个算法来重建这个队列。

注意：
总人数少于1100人。

示例

解法一：贪心算法+从小到大+留空位

第一种解法是可以考虑，将人们按照身高从低到高排序，对于身高低的人，他的 (h, k) 中小于 h 的身高的人已经落座了，他只需要考虑大于等于 h 的人对他的影响即可，那么这些大于等于 h 身高的人都要在后面落座，也就是罗坐在他左边的空位中。因此，对于每个人的 k ，他只需要在这一次遍历中，在前面留出来 k 个空座即可保证他前面有 k 个大于等于他的身高的人了。这里面 sort 其实不用 lambda 表达式也可以。

解法二：贪心算法+从大到小+直接插入

第二种解法是跟第一种相同的思路，只不过是从大到小反过来思考。对于身高最高的人，他可以直接插进去，后面的人都比他们矮，因此怎么插队都不会影响到高个子的人。对于后面身高不如他们的人们，比他个高的都已放置，比他个低的都未放置，而只有比他个高的对他的 k 值有影响，比他个低的对他的 k 值无影响，所以，只需要把他放置到当前队列的第 k+1 位即可。

排序完：
[[7,0], [7,1], [6,1], [5,0], [5,2], [4,4]]
插入的过程：
插入[7,0]：[[7,0]]
插入[7,1]：[[7,0],[7,1]]
插入[6,1]：[[7,0],[6,1],[7,1]]
插入[5,0]：[[5,0],[7,0],[6,1],[7,1]]
插入[5,2]：[[5,0],[7,0],[5,2],[6,1],[7,1]]
插入[4,4]：[[5,0],[7,0],[5,2],[6,1],[4,4],[7,1]]

优化⭐

注意，此处并未直接将 people[i] 送入 res 数组中，而是先将下标 i 送入 idx 数组中，待 idx 位置最终排好之后再对 res 数组进行赋值。
这是因为，vector 的 insert 操作的时间复杂度为 o(n) 级别，其中包含了大量的内存整块迁移的操作。如果vector 的 vector 来回迁移的话，会非常慢。而 int 的 vector 的操作会省很多时间。

确实快了很多，上面是 380ms ，这个是 44ms 。
