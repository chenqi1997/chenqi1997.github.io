617. 合并二叉树

题目：

给定两个二叉树，想象当你将它们中的一个覆盖到另一个上时，两个二叉树的一些节点便会重叠。

你需要将他们合并为一个新的二叉树。合并的规则是如果两个节点重叠，那么将他们的值相加作为节点合并后的新值，否则不为 NULL 的节点将直接作为新二叉树的节点。

示例 1:

注意: 合并必须从两个树的根节点开始。

解法：DFS

看到树还是想到用递归来解决子问题，再拼接到根节点上，这道题一开始的想法偏了一些，把DFS想复杂了。其实就简单的策略：

  1. 当前 t1 为空，则 new 一个 t2 节点返回。
  2. 当前 t2 为空，则 new 一个 t1 节点返回。
  3. 同为空在前面包含了，返回的是 nullptr。
  4. 同不为空则 new 一个两个节点值之和的节点。
