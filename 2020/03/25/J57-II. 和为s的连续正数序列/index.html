J57-II. 和为s的连续正数序列

题目：

输入一个正整数 target ，输出所有和为 target 的连续正整数序列（至少含有两个数）。

序列内的数字由小到大排列，不同序列按照首个数字从小到大排列。

解法二：⭐双指针法

还是可以维护一个由双指针围起来的滑动窗口，在这里对于滑动窗口有几种讨论情况：

  1. 当窗口的和小于 target 的时候，窗口的和需要增加，所以要扩大窗口，窗口的右边界向右移动
  2. 当窗口的和大于 target 的时候，窗口的和需要减少，所以要缩小窗口，窗口的左边界向右移动
  3. 当窗口的和恰好等于 target 的时候，我们需要记录此时的结果。设此时的窗口为 [i, j)，那么我们已经找到了一个 i开头的序列，也是唯一一个 i 开头的序列，接下来需要找i+1 开头的序列，所以窗口的左边界要向右移动。

注意❗：这里面其实还有一点细节可以优化，就是在每一个 sum == target的时候，其实可以直接 l += 2，因为可以证明如果 l 开头的序列和等于 target， l + 1开头的序列和必不等于 target。不过其实没差多少-。-
