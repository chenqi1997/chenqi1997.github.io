J14-I&II*. 剪绳子/ Leetcode 343

题目：

给你一根长度为 n 的绳子，请把绳子剪成整数长度的 m 段（m、n都是整数，n > 1并且m > 1），每段绳子的长度记为 k[0],k[1]…k[m] 。请问 k[0]*k[1]*…*k[m] 可能的最大乘积是多少？例如，当绳子的长度是8时，我们把它剪成长度分别为2、3、3的三段，此时得到的最大乘积是18。

示例 1：

输入: 2
输出: 1
解释: 2 = 1 + 1, 1 × 1 = 1

示例 2:

输入: 10
输出: 36
解释: 10 = 3 + 3 + 4, 3 × 3 × 4 = 36

提示：

2 <= n <= 58

解法一：动态规划

首先最容易想到的是复杂度为O(n2)的动态规划解法，维护一个一维的 dp 数组，dp[i]表示数字 i 拆分成至少两个正整数的和的最大乘积， dp 数组大小为 n + 1，初始值全部设为 1。由于 n = 1, 2的时候 dp[n] 均为1，因此我们可以设置 i 从3到 n 遍历。在循环 j 中，对于小于 i 的 j，我们按照两种情况进行讨论：如果是拆分成两个数字，即为 j * (i - j)，如果是拆分成多个数字，由于我们是遍历 j ，因此要考虑将 i - j 拆分，这样才可以避免重复，因此构造的动态规划状态转移方程为：

⭐解法二：数学解法+贪心规则

通过数学计算可以得到贪心规则如下：

最高优先级： 3 。把绳子尽可能切为多个长度为 3 的片段，留下的最后一段绳子的长度可能为 0,1,2 三种情况。
次高优先级： 2 。若最后一段绳子长度为 2，则保留，不再拆为 1+1 。
最低优先级： 1； 若最后一段绳子长度为 1；则应把最后的 3 + 1 替换为 2 + 2，因为2×2 > 3×1。

算法流程：
当 n≤3 时，按照贪心规则应直接保留原数字，但由于题目要求必须剪成 m>1 段，因此必须剪出一段长度为 1 的绳子，即返回 n - 1 。
当 n > 3 时，求 n 除以 3 的 整数部分 a 和 余数部分 b （即 n = 3a + b ），并分为以下三种情况：

  1. 当 b = 0 时，直接返回 $3^a$.
  2. 当 b = 1 时，要将一个 1+3 转换为 2+2，因此返回 $3 ^{a-1} × 4​$.
  3. 当 b = 2 时，返回 $3^a ×2$。

时间空间复杂度都为O(1)。
