20. Valid Parenthesis/有效的括号

给定一个只包括 ‘(‘，’)’，’{‘，’}’，’[‘，’]’ 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

示例 1:

输入: “()”
输出: true

示例 2:

输入: “()[]{}”
输出: true

示例 3:

输入: “(]”
输出: false

示例 4:

输入: “([)]”
输出: false

示例 5:

输入: “{[]}”
输出: true

解法

这个是之前做过的一个很经典的用到栈的题，主要逻辑是利用栈顶的push() top()与 pop()函数来对 s[i] 元素进行操作，如果是左括号( [ { 则依次 push() 到栈里面，如果是右括号就 push栈顶的元素看是不是匹配，如果匹配就用 pop() 函数来删除栈顶的左括号，如果不匹配直接return false。

⭐题目里面没有考虑 () [] {} 三者之间的包含关系情况，也就是输入 ( [ { } ] ) 返回的是true。

⭐复习有关stack STL的函数如push / top / pop / empty / size 等等。
