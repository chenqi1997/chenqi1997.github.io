912. Sort an Array/排序数组

题目：

给你一个整数数组 nums，将该数组升序排列。

示例 1：

示例 2：

提示：

  1. 1 <= nums.length <= 50000
  2. -50000 <= nums[i] <= 50000

C++ std sort：整体采用快速排序的算法，并且选择哨兵的时候，采用了三数取中法（头部，中部，尾部元素，三者的中间值作为哨兵）。如果在某次递归时，数据量小于16个，则采用插入排序。如果递归层次过深，会采用桶排序。

解法一：堆排序 时间 O(NlogN) 空间 O(N)

这道题一上来的想法是用快排肯定会有测试样例是最坏的 O(N^2) 复杂度超时，来一手堆排序，果然没超时。

解法二：归并排序 时间 O(NlogN) 空间 O(N)

解法三：⭐⭐计数排序

时间复杂度 O(N + V)，N为元素个数，V 为元素的取值范围。
空间复杂度 O(V)，借助 hash map 等容器可以变为 O(N)。
实现思路，遍历数组 nums, 统计每个数字出现的次数，存储到 count 中。记最小数字为 min，最大数字为 max。 从 min 到 max 枚举, 按 count[i] 将 i放回 nums。如下图所示：
