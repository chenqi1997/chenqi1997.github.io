32*. Longest Valid Parenthese/最长有效括号

题目：

给定一个只包含 '(' 和 ')' 的字符串，找出最长的包含有效括号的子串的长度。

解法：⭐⭐⭐⭐动态规划

动态规划问题的三部曲：

  1. 确定状态
  2. 确定状态转移方程
  3. 确定边界情况

首先需要确定状态，这里面我们维护一个dp[i]数组，其中 i 表示在到第 i 个字符的时候能构成的最长有效括号的长度。接下来确定状态转移方程，这道题跟 Leetcode 22 生成括号 的动态规划实现方法比较像，也是要根据每次遍历的 s[i] 是左括号还是右括号来讨论。这里面如果 s[i] == ‘(‘ 我们不会对应生成 dp[i] ，因为这个是不能构成合法的括号的。接下来再讨论 s[i] == ‘)’ 情况下的两种情况：

  1. s[i - 1] == '(' ，也就是字符串形如 “……()”，我们可以推出：
    dp[i] = dp[i − 2] + 2。
    因为结束部分的 "()" 是一个有效子字符串，并且将之前有效子字符串的长度增加了 2.
  2. s[i - 1] == ')'，也就是字符串形如 “.......))”，我们可以推出：
    if s[i - dp[i - 1] - 1] == '(' ，dp[i] = dp[i − 1] + dp[i − dp[i − 1] − 2] + 2。
    因为如果倒数第二个 ) 是一个有效子字符串的一部分（记为 subs ），我们此时需要判断 subs 前面一个符号是不是 ( ，如果恰好是 ( ，我们就用 subs 的长度 (dp[i - 1) 加上 2 去更新 dp[i] 。除此以外，因为之前前面的 ( 的存在，dp[i − dp[i − 1] − 2] 并没有跟后面的最长有效括号连上，也就相当于没有计算进去，因此我们也会把子字符串 subs 前面的有效字符串的长度加上，也就是 dp[i − dp[i − 1] − 2] .
