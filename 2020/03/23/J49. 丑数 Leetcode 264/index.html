J49. 丑数 / Leetcode 264

题目：

我们把只包含因子 2、3 和 5 的数称作丑数（Ugly Number）。求按从小到大的顺序的第 n 个丑数。

示例:

输入: n = 10
输出: 12
解释: 1, 2, 3, 4, 5, 6, 8, 9, 10, 12 是前 10 个丑数。

说明:

1 是丑数。
n 不超过1690。

解法一：动态规划+指针

这里面 p2 , p3 , p5 分别是指向前面已经出现过的 dp[p2]，dp[p3]，dp[p5]。他们的作用是保存分别乘以2 ,3, 5还没有出现的丑数。如果新的 dp[i] == dp[p2] * 2，则说明这个 dp[p2] 前面的丑数 × 2肯定是已经出现过的了，所以需要 p2++。这样，通过三个指针不停地从前向后指，可以不断更新最新的 dp[i] 的值。这道题也算是借鉴了一点点 双指针 的思想？这个思想也可以看成是三个有序链表的无重复合并问题。

⭐注意，这里面每个for循环中的三个if语句不是只能同时调用一个的，比如1, 2, 3, 4, 5, 6, 8, 9, 10。这里面在 i = 5的时候dp[i] = 6，是同时满足dp[i] == dp[p2]*2 ，dp[i] == dp[p3]*3的，因此p2 p3都需要++，从而避免了存入重复丑数的情况。

注意这里面用 min() 函数只能两个两个比较，也可以用 C++11 的新特性：

min({dp[p2] * 2, dp[p3] * 3, dp[p5] * 5});
