J53-I. 在排序数组中查找数字 I / Leetcode 34

题目：

统计一个数字在排序数组中出现的次数。

示例 1:

输入: nums = [5,7,7,8,8,10], target = 8
输出: 2

示例 2:

输入: nums = [5,7,7,8,8,10], target = 6
输出: 0

限制：

0 <= 数组长度 <= 50000

解法一：二分查找

显然直接用O(n)的查找太捞了，想到用二分查找来进行查找，由于整个数组是排序过的，所以可以分别控制if条件，使得当前的left指针分别通过两次循环指向连续的target的左端和右端，即可通过这个范围求出次数。
