166*. Fraction to Recurring Decimal/分数到小数

题目：

给定两个整数，分别表示分数的分子 numerator 和分母 denominator，以字符串形式返回小数。

如果小数部分为循环小数，则将循环的部分括在括号内。

解法：模拟除法+哈希表找循环

这道题思路很简单，但是要想一点问题都没有比较困难，我这个花了很久的时间，一次一次试错才写出来最终的解答。。吐了🤮测试样例真难满足啊。。

主要踩过的错误如下：

  1. 如果直接整除了则不需要添加 .
  2. 1/6 = 0.1(6)，因此需要一个参数 idx 定位开始进入循环的点，如果 idx == 0 则直接 res += '(' + totalnum + ')'; 否则，需要 res += totalnum.substr(0, idx) + '(' + totalnum.substr(idx) + ')'; 先把不循环的字符串加上，再把循环的部分字符串放到括号内加到后面。
  3. 负号的问题，直接在前面判断一下加上，然后对分子分母同取绝对值。
  4. 分母为 0 直接返回 0。
  5. 1/-2147483648 这个样例无法取绝对值，会超出 int 范围，同时在模拟除法计算的时候也会 overflow，因此需要把 numerator 和 denominator 进行强制类型转换，模拟除法的余数也应该是 long long 类型的，后来烦了干脆把所有的参数都设置为 long long 类型。
