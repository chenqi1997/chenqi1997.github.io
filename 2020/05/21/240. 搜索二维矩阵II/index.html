240. Search a 2D Matrix II/搜索二维矩阵II

题目：

编写一个高效的算法来搜索 m x n 矩阵 matrix 中的一个目标值 target。该矩阵具有以下特性：

  • 每行的元素从左到右升序排列。
  • 每列的元素从上到下升序排列。

示例:

现有矩阵 matrix 如下：

给定 target = 5，返回 true。

给定 target = 20，返回 false。

解法一：从右上角开始遍历

这道题很显然的一个困境是，如果从左上角开始遍历的话，那么其实每次判断完当前位置和 target 的大小关系之后，是很难做出选择的，因为它右边和下面的元素都比它大，不知道该走哪边，所以我们自然想到从右上角走，如果 target 比这个值大那么就向下走，如果比这个值小那么久向左走。直到碰到边界才会返回 false 。

时间复杂度为 O(m + n) 。

解法二：二分法

看到有序，第一反应就是二分查找。最直接的做法，一行一行的进行二分查找即可。

此外，结合有序的性质，一些情况可以提前结束。

比如某一行的第一个元素大于了 target ，当前行和后边的所有行都不用考虑了，直接返回 false 。

某一行的最后一个元素小于了 target ，当前行就不用考虑了，换下一行。

时间复杂度的话，如果是 m 行 n 列，就是 O(mlog(n))。
