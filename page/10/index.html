J28. 对称的二叉树 / Leetcode 101

题目：

请实现一个函数，用来判断一棵二叉树是不是对称的。如果一棵二叉树和它的镜像一样，那么它是对称的。

例如，二叉树 [1,2,2,3,4,4,3] 是对称的。

    1
   / \
  2   2
 / \ / \
3  4 4  3

但是下面这个 [1,2,2,null,3,null,3] 则不是镜像对称的:

  1
 / \
2   2
 \   \
 3    3

示例 1：

输入：root = [1,2,2,3,4,4,3]
输出：true

示例 2：

输入：root = [1,2,2,null,3,null,3]
输出：false

限制：

0 <= 节点个数 <= 1000

解法一：利用J27里面的 mirrorTree（太笨了！！）

很烦，刚做完27题所以想着这道题不如就先写一个 mirrorTree得到对称之后的树，再写一个 equalTree 来判断一下是否相等就行，但是明明感觉逻辑没什么问题但是代码就是会错，排查了半天也没排查明白，先占个坑TAT

解法二：递归思想

如果递归来看，对于目前的手上的两个节点 p 和 q，对称即为 p 的左子节点的值和 q 的右子节点的值相等，且 p 的右子节点的值与 q 的左子节点的值相等。这里面为了方便递归，我们先给出 root 的左右节点，将它们分别作为新定义的重载函数 isSymmetric 的输入。

解法三：迭代思想/BFS

利用队列来进行迭代，每次提取两个节点并且比较它们的值，然后同时将两个节点的左右子节点按相反的顺序插入到队列中，直到队列中不存在元素或者树不对称时结束。

J27. 二叉树的镜像 / Leetcode 226

题目：

请完成一个函数，输入一个二叉树，该函数输出它的镜像。

例如输入：

     4
   /   \
  2     7
 / \   / \
1   3 6   9

镜像输出：

     4
   /   \
  7     2
 / \   / \
9   6 3   1

示例 1：

输入：root = [4,2,7,1,3,6,9]
输出：[4,7,2,9,6,3,1]

限制：

0 <= 节点个数 <= 1000

解法一：递归法

遇到树最先想到的想法是做递归，通过不断进行递归向下完成树的从上到下的旋转。在每一层的旋转过程中都借助一个 tmp 指针来交换传递左右指针，最后返回根节点。

⭐解法二：非递归法：队列实现

递归的实现方法可以通过队列的先入先出性质来实现，在遍历节点的过程中，先放入根节点 root，再取出头节点并交换左右节点，如果左右节点存在，再存入队列中并且依次取出，从而不断向下遍历。

注意 queue 队列类没有 top 函数，有的是 front , back , pop , empty , size 。不要跟栈弄混了。

J26. 树的子结构

题目：

输入两棵二叉树A和B，判断B是不是A的子结构。(约定空树不是任意一个树的子结构)

B是A的子结构， 即 A中有出现和B相同的结构和节点值。

例如:
给定的树 A:

    3
   / \
  4   5
 / \
1   2

给定的树 B：

返回 true，因为 B 与 A 的一个子树拥有相同的结构和节点值。

示例 1：

输入：A = [1,2,3], B = [3,1]
输出：false

示例 2：

输入：A = [3,4,5,1,2], B = [4,1]
输出：true

限制：

0 <= 节点个数 <= 10000

解法：

如果树 B 是树 A 的子结构，那么树 B 根节点一定是树 A 的某一个子节点，这里面对于二叉树的遍历分为两步：

  1. 先序遍历树 A 中的每个节点，递归调用 isSubStructure 函数，来判断当前 A 的节点是否为 B 的根节点。
  2. 如果已经是 B 的根节点了，递归调用 recur 函数来进行 DFS 搜索，在搜索过程中判断 B 是否与找到的 A 的子树有相同的结构和节点值，这里面搜索的过程判断的应该是 当前节点值相等，再调用左右子树做 DFS 搜索。
